<h1 align="center">WebDisplays Brightness Control</h1>

<p align="center"><img width=300px alt="Mod Icon" src="https://i.imgur.com/EoFC3um.png"></p>

**WebDisplays Brightness Control** is an addon for the [WebDisplays](https://www.curseforge.com/minecraft/mc-mods/webdisplays) mod, allowing you to adjust the brightness of the in-game screens.

This addon makes WebDisplays screens better usable with shaders, so you can avoid blinding yourself or others while keeping the visuals intact. All you have to do is set the brightness config to your preference, and it's done.

## Features

- Adjustable screen brightness.
